Release Notes

Clip 的每一次更新，都为了更好的体验

v2.0 Latest

2026-06-12

1.0 版本在 SwiftUI 框架下已趋于稳定，为了追求更极致的性能和体验，我决定对整个应用进行彻底重构。2.0 版本采用 AppKit 架构重写，在启动速度、流畅度和响应速度方面都有显著提升，带来更加流畅的使用体验。

What's new

  • 新增 MCP 支持（Beta）
  • 历史记录列表完整支持多选操作 #16
  • 支持直接拖拽文本、文件等到剪贴板
  • 更丰富的卡片菜单
  • 初步适配 macOS 27

Improvements

  • 优化识别复制的内容，比如在 IntelliJ IDEA 复制文件，现在能准确识别
  • 最低支持版本降低为 macOS 14
  • 修复了一些已知问题

v1.1

2026-05-12
注意：应用启动没有数据是因为该版本修改了数据存储位置，避免每次启动都提示文稿授权弹窗， 只需在 设置 – 存储页面导入旧数据即可，旧数据存储位置：文稿 – Clip – Clip.sqlite3， 后续可删除旧数据，或保留备份

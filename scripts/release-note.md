# Release Notes

## 🎉 0.3.1 (2026-01-23)

> **⚠️ 重要提示：**
> 本版本包含应用沙盒迁移更新，避免频繁权限弹窗。
> 1. 首次启动时，应用将提示导入旧的偏好与设置，请按照提示选择com.crown.clipboard文件后可完成配置迁移（新用户可跳过）。
> 2. 应用启动后可到设置页面数据栏目点击导入按钮选择Clip/Clip.sqlite3进行旧数据导入，否则数据可能会丢失。

### What's new
- 图片内容支持搜索（仅限于新的图片历史内容），预览支持OCR
- 设置页面新增数据备份、导入，日志查看

### Bug Fixes
- 修复了一些已知问题

---

## 🎉 0.3.0 (2026-01-01)

### What's new
- 提供按类型、应用或日期更丰富的过滤方式，以便于更好的查询历史记录
- 关键字搜索支持高亮展示
- 文本支持编辑并保存
- 支持暂停并恢复剪贴板记录

### Improvements
- 改进链接、图片卡片展示
- 预览窗口体验优化
- 设置-关于界面的图标支持直接拖拽获取权限
- 更好的检测应用更新
- 兼容macOS Sonoma

### Bug Fixes
- 修复了一些已知问题
